
# basic text editor shortcuts:

## ctrl modifer
- moving back one word  
    `ctrl + (left/right) arrowkey `

- deleting one word  
    `ctrl + backspace`

- selecting one word  
   `ctrl + shift + (left/right) arrowkey`


# vscode shortcuts:

## moving lines
- `alt + (up/down) arrowkey`

## copying lines
- `alt + shift + (up/down) arrowkey`

## indenting | un-indenting
- `ctrl + /`

## spawning cursor
- `alt + left click`
- `ctrl + alt + shift + (up/down) arrowkey`

## multi line selections
- `middle mouse button`

## going to definition
- `ctrl + left click`

## copy line
- `ctrl + c`

## cut line (useful for deleting a line)
- `ctrl + x`

# debugger shortcuts:

## continue (to next breakpoint)
- `f5`

## stop (debugging)
- `shift + f5`

## step over (next line)
- `f10`

## step in (go into function)
- `f11`

## step out (leave function)
- `shift + f11`
